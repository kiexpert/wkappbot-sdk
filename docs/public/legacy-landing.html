Features

왜 WKAppBot인가?

🔍

5단계 요소 탐색

UIA → Vision Cache → OCR → Vision API → 좌표 방식으로 어떤 UI도 찾아냅니다.

🌐

CDP 브라우저 자동화

Chrome DevTools Protocol로 JavaScript 렌더링 페이지도 완벽하게 제어합니다.

🔒

관리자 권한 자동화

--sudo 옵션으로 UAC 승격 없이 HTS · 증권사 등 관리자 앱을 자동화합니다.

🎙️

WhisperRing 음성 입력

실시간 음성 인식으로 모든 앱에 포커스리스 텍스트 입력을 지원합니다.

🤖

AI 통합

GPT · Gemini · Claude 3-way triad 분석으로 복잡한 자동화 판단을 AI에게 위임합니다.

⚡

Eye 데몬

항상 실행 중인 Eye 프로세스로 명령 실행 지연 없이 즉시 자동화를 수행합니다.

How to Buy

구매 방법

1

GitHub 계정 준비

WKAppBot 라이선스는 GitHub 계정으로 관리됩니다. github.com에서 계정을 만드세요.

2

입금

원하는 플랜 금액을 입금하고 입금자명에 GitHub ID를 적어주세요.
계좌번호는 문의 시 안내드립니다.

3

자동 권한 부여

입금 확인 후 kiexpert/wkappbot-sdk 레포에 collaborator로 초대됩니다. GitHub 알림을 확인하세요.

4

gh CLI 인증 후 사용

gh auth login 으로 인증하면 WKAppBot이 자동으로 권한을 인식합니다.
